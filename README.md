# Bone Marrow Survival Prediction

Projeto de Machine Learning para análise de dados clínicos e previsão de sobrevivência de pacientes.

## Descrição

Este projeto implementa um pipeline de Machine Learning utilizando Python e Scikit-learn.  
O objetivo é processar dados clínicos, preparar as variáveis e treinar um modelo capaz de realizar previsões com base nessas informações.

## Arquivos do projeto

- `bone_marrow_pipeline.py`  
Script principal responsável pelo carregamento dos dados, pré-processamento e treinamento do modelo.

- `bone-marrow.arff`  
Dataset utilizado no projeto, contendo informações clínicas dos pacientes.

## O que o código faz

- Carrega os dados no formato `.arff`
- Realiza tratamento e organização dos dados
- Separa variáveis numéricas e categóricas
- Aplica transformações como codificação e normalização
- Treina um modelo de Machine Learning
- Executa o pipeline completo de processamento e previsão

## Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Scikit-learn
- SciPy
